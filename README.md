# Diego García - Web Oficial

## Descripción
Proyecto web desarrollado para el piloto de Autocross Diego García. La interfaz funciona como portafolio profesional y punto de contacto, estructurando la información sobre su palmarés, noticias, galería multimedia y especificaciones técnicas de su vehículo. El diseño la adaptabilidad a dispositivos móviles y el cumplimiento normativo de accesibilidad.

## Tecnologías Utilizadas
* HTML5 (estructuración semántica)
* CSS3 y Tailwind CSS
* JavaScript

## Características Principales
* Interfaz responsiva adaptada a múltiples resoluciones de pantalla.
* Navegación dinámica con menú específico para dispositivos móviles.
* Carrusel de imágenes interactivo.
* Integración de vídeo de fondo con soporte para subtítulos integrados.
* Formulario de contacto funcional en la parte cliente.

## Accesibilidad (WCAG 2.1 - Nivel AA)
El desarrollo incluye las siguientes implementaciones técnicas para garantizar la accesibilidad:

* Etiquetas WAI-ARIA para contextualizar elementos dinámicos a las tecnologías de asistencia.
* Enlaces de salto (skip links) y gestión visible del foco para la navegación estricta por teclado.
* Áreas de interacción táctil dimensionadas a un mínimo de 44x44 píxeles.
* Detección y aplicación de la regla prefers-reduced-motion para usuarios con sensibilidad al movimiento.
* Jerarquía estricta de encabezados y alternativas textuales en la totalidad del contenido multimedia.

## Instrucciones de Ejecución
Para desplegar el proyecto en un entorno local, ejecute la siguiente sintaxis en su terminal:

git clone [https://github.com/dani285i/Diego-Garcia-Tailwind.git](https://github.com/dani285i/Diego-Garcia-Tailwind.git)
cd Diego-Garcia-Tailwind/src
A continuación, abra el archivo index.html resultante en cualquier navegador web moderno. No se requiere la instalación de dependencias en el servidor al utilizar Tailwind mediante CDN.
